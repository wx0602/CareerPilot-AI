# 团队协作流程

## 第一阶段集成顺序

1. B 和 D 先确认 API 字段、AI JSON 字段。
2. A 使用假数据完成页面壳子。
3. B 搭建后端接口框架。
4. C 准备题库、RAG 和材料解析。
5. D 准备 Prompt、Agent、评分和报告 JSON。
6. C 将检索结果交给 D。
7. D 将 AI 问题、评分、报告结构交给 B。
8. B 将接口交给 A 接入页面。

## 第二阶段集成顺序

1. A 扩展数字人页面、语音录制和音频播放。
2. B 扩展 ASR、TTS、历史记录和部署接口。
3. C 优化题库标签、RAG、PPT 和计划书解析。
4. D 优化追问、评分、报告和模式切换。
5. 全员完成部署演示。

## README 要求

每个人在自己目录维护总结 README：

- 做了什么
- 输入输出字段
- 如何运行
- 如何和其他人集成
- 当前未完成项
