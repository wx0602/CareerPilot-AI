# 阶段计划

## 第一阶段

目标：跑通基础流程。

范围：

- 登录
- 上传简历 / JD
- 企业笔试
- 自动批改
- 错题分析
- 文本面试
- AI 追问
- 简单报告

交付：

| 人 | 任务 | 交付物 |
| --- | --- | --- |
| A | 登录页、上传页、笔试页、文本面试页、报告页 | 可点击的前端页面 |
| B | 登录、上传、笔试、面试、报告接口和数据库 | API 文档 + 数据库 |
| C | 题库、导入、简历/JD 解析、简单检索 | 题库数据 + 检索函数 |
| D | Skill、Agent、Prompt、评分逻辑 | AI 函数 + JSON 输出格式 |

## 第二阶段

目标：扩展演示效果和完整体验。

范围：

- 数字人展示
- 语音提问
- 语音回答
- 报告美化
- 历史记录
- 部署演示

交付：

| 人 | 任务 | 交付物 |
| --- | --- | --- |
| A | 数字人页面、语音录制、音频播放、报告美化 | 数字人交互页面 |
| B | ASR/TTS 接口、历史记录接口、部署 | 语音接口 + 部署地址 |
| C | 优化题库标签、RAG、解析 PPT/计划书 | 更准的检索结果 |
| D | 优化追问、评分、报告、模式切换 | 完整报告 + 模式切换 |
