# C 工作区：题库、RAG、文档解析

本目录归 C 负责，当前只提供目录和数据格式占位，不包含解析、检索或向量库逻辑。

## 建议职责

- 准备企业、岗位、难度、题型维度的题库
- 将题库导入可检索结构
- 解析简历、岗位 JD、项目介绍、商业计划书、路演 PPT
- 为 D 的 Agent 提供检索结果

## 目录说明

| 目录 | 用途 |
| --- | --- |
| `question_bank/` | 原始题库、题库清洗结果、题库导入说明 |
| `rag/` | RAG 流程、检索函数、召回结果格式 |
| `parsers/` | 简历、JD、PPT、计划书解析入口 |
| `samples/` | 题库和材料字段示例 |
| `chroma_store/` | 本地 Chroma 数据目录，不提交真实数据 |
